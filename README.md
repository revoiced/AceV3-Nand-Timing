# AceV3-Nand-Timing
Ported Phat timing files to support more Multi-Nand options on the Ace V3 Chip.

Credits to Josh(Octal450) and DrTrinity for source code and being a helping hand.
https://github.com/Octal450/RGH1.2-V2-Slim

Pins:

Nand1 = R8

Nand2 = R7

Nand3 = R6

Nand4 = R5

Nand5 = R4

SMC = R3

Button = R10
